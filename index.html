Frogger - The Classic Arcade Game

As part of the Udacity Frontend Nanodegree, students are given the building blocks needed to develop a Frogger clone. This project gives experience working with the canvas and working with object-oriented JavaScript.

You can test the game here: http://silentvibe.com/frogger/
